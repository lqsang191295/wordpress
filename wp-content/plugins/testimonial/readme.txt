== Description ==

Testimonial is pure html & css3 responsive Testimonial slider for wordpress. Very easy to use via shortcode anywhere,

<strong>Plugin Features</strong><br />


* Fully responsive and mobile ready.
* Unlimited Testimonial anywhere.
* Slider & Grid View.
* Query Post from Any Post Type, your previous testimonial posy type.
* Slider Navigation & Pagination.
* Slider Navigation different 3 Positions & different Theme for Navigation.
* Custom Color for Pagination & different Theme for Pagination.
* Ready 4 layout.
* Create your own layout by filter hook.
* Custom number of Column for different device.
* Custom margin grid items.
* Custom background image, color & padding for container.





== Installation ==

1. Install as regular WordPress plugin.<br />
2. Go your plugin setting via WordPress Dashboard and find "<strong>Testimonial</strong>" activate it.<br />

After activate plugin you will see "Product Slider" menu at left side on WordPress dashboard click "New Testimonial Showcase" and use the options field "Testimonial"<br />

<br />
<strong>How to use on page or post</strong><br />
When Testimonial options setup done please publish Testimonial showcase as like post or page<br />

and then copy shortcode from top of <strong>Testimonial Options</strong> `[testimonial id="5066"]`<br />

then paste this shortcode anywhere in your page to display grid<br />
